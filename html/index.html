Research context (dissertation)

Evaluating Progressive Web Applications as a Viable Alternative to Native Mobile Applications — TaskFlow is a paired implementation (this PWA and a native Android build) used to compare accessibility affordances, deployment friction, offline behaviour, and inclusive design patterns in the browser versus a platform-specific stack.

Use Profile → Settings for install, readability, and contrast options demonstrated in the PWA version.
